Array, içeriğine [dizin](/index-terimi)le (satır numarasıyla) erişilen bir veri grubu oluşturmak için kullanılan bir [data structure](/data-structure)'dır.

Örnek olarak;

```js
// Array oluşturmak
var myArray = ['a', 'b', 'c'];

// Array'den bir elemanı okumak
myArray[1]; // 'b'
```
