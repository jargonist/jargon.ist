Array, içeriğine dizinle(satır numarasıyla) erişilen bir veri grubu oluşturmak için kullanılan bir [data structure](/data-structure)dır.

Ornek olarak;
```javascript
// Array oluşturmak
var myArray = ['a', 'b', 'c'];

// Arrayden bir elemanı okumak
myArray[1] // 'b'
```
